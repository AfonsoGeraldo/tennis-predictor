# Tennis Predictor 🎾

Projeto pessoal de Machine Learning para prever vencedores de jogos de ténis ATP, com base em dados históricos.

## Objetivo
Explorar Machine Learning na prática, usando dados de ténis (ranking, superfície, head-to-head) para prever resultados de jogos.

## Estrutura do projeto
- `data/` — datasets usados no projeto
- `notebooks/` — exploração e experimentação
- `src/` — código Python reutilizável
- `requirements.txt` — dependências do projeto

## Como correr
1. Criar ambiente virtual: `python -m venv venv`
2. Ativar: `venv\Scripts\activate` (Windows)
3. Instalar dependências: `pip install -r requirements.txt`
